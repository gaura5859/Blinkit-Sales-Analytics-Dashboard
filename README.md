# Blinkit Sales & Outlet Performance Dashboard

## Project Overview

This project analyzes Blinkit's sales performance, outlet distribution, and product category trends using Microsoft Excel.

The dashboard provides business insights through interactive visualizations, KPI cards, slicers, and pivot-based reporting.

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting
* Dashboard Design

---

## Key KPIs

* Total Sales
* Average Sales
* Number of Items
* Average Ratings

---

## Dashboard Features

### Sales Analysis

* Total Sales Tracking
* Average Sales Monitoring
* Product Category Performance

### Outlet Analysis

* Outlet Size Distribution
* Outlet Location Analysis
* Outlet Establishment Trends

### Product Insights

* Item Type Performance
* Fat Content Analysis
* Outlet-wise Sales Comparison

---

## Business Insights

* Tier 3 outlets generated the highest sales.
* Fruits & Vegetables and Snack Foods were top-performing categories.
* High-sized outlets contributed significantly to total revenue.
* Sales trends varied across outlet establishment years.
